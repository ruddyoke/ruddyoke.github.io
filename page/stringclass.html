StringClass

Concept

Permettre d'utiliser les fonctions string de php de façon orienté-objet.

Cette classe se base sur les meilleurs fonctions string en terme de rapidité. Plus besoin de lire les fonctions du manuel PHP pendant des heures avant de savoir quelle fonction fait quoi.

Si vous avez déjà utilisé JavaScript ou java alors la prise en main sera facile .

First step

Utiliser composer pour créer le fichier autoload et les bibliothèques externes.

Si vous avez déjà utilisé JavaScript ou java alors la prise en main sera facile .

Aperçu Rapide

require '../vendor/autoload.php';
use Roke\String;

$a = new String(" mon string ");
echo $a->bold(); // ouput mon string
echo $a->transform->ucfirst(); // ouput mon string
echo $a->length();
